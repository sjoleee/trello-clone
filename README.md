<div align="center"><h1> 칸반보드 </div>

<br>
- React와 TypeScript에 익숙해지기 위해 진행한 개인 프로젝트에요.<br>
- Recoil로 상태관리, Beautiful-DnD로 드래그 앤 드롭을 구현했어요.<br>
- 기본적인 투두리스트 앱을 만들었고, Node.js를 활용하여 DB와 연결하기 위해 준비중이에요.<br>

<br>
  
## :rocket: 기술 스택

- React
- TypeScript
- Recoil
- styled-components
- Beautiful-DnD
- React-hook-form

<br>
